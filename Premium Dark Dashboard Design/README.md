# VapeStore - Премиальный Dark Dashboard

Мобильное приложение для управления магазином электронных сигарет в премиальном тёмном стиле с пастельными акцентами.

## 🎨 Дизайн

- **Стиль**: Премиальный темный финансовый дашборд (вдохновлен Cuberto/Ghulam Rasool)
- **Цветовая схема**: Графитовый фон (#0F1115) + пастельные акценты (mint, blue, lavender, pink)
- **UI компоненты**: Крупные скругления (18-28px), тонкие outline-иконки, плавные анимации

## 📱 Разделы приложения

### 1. Главная (Home)
- KPI метрики (продажи, выручка, прибыль)
- График прибыли за неделю
- Быстрые действия

### 2. Приёмка (Accept)
- Сканирование штрих-кодов
- Ручной ввод товаров
- История приёмок

### 3. Продажа (Sell)
- Поиск и фильтрация товаров
- Корзина с мгновенным расчетом
- Выбор способа оплаты (наличные/карта/раздельно)
- Скидки

### 4. Кабинет (Cabinet)
Три вкладки:
- **Склад**: Сводка по остаткам, группировка по категориям
- **Продажи**: Аналитика по периодам, список продаж
- **Товары**: Управление каталогом, редактирование товаров

### 5. Управление (Management)
- Управление продажами (редактирование, отмена)
- Долги клиентов
- Резервы товаров
- Платёжные карты
- Резервное копирование данных

## 🛠 Технологии

- **React** 18.3.1
- **React Router** 7.13.0 - навигация
- **Motion** (Framer Motion) 12.23.24 - анимации
- **Tailwind CSS** 4.x - стилизация
- **Lucide React** - иконки
- **Recharts** - графики
- **Sonner** - уведомления

## 🎯 Ключевые особенности

### UX/UI
- ✅ Плавные анимации появления (fade/slide 150-250ms)
- ✅ Press-эффекты на карточках (scale 0.98)
- ✅ Анимированный индикатор активной вкладки
- ✅ Скелетоны для состояний загрузки
- ✅ Toast-уведомления для обратной связи

### Функциональность
- ✅ Полная навигация между разделами
- ✅ Поиск и фильтрация товаров
- ✅ Корзина с управлением количеством
- ✅ Множественные способы оплаты
- ✅ Управление остатками и продажами

### Доступность
- ✅ ARIA labels на всех icon buttons
- ✅ Контрастность текста (WCAG AA)
- ✅ Крупные тач-зоны (минимум 44x44px)
- ✅ Focus indicators
