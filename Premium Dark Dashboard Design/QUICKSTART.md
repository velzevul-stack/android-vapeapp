# VapeStore - Быстрый старт

## 🚀 Запуск приложения

Приложение готово к использованию сразу после открытия в Figma Make.

## 📱 Навигация

Используйте нижнюю панель навигации для перехода между разделами:

1. **Приёмка** - Добавление товара на склад
2. **Продажа** - Оформление продаж
3. **Главная** (центральная) - Дашборд с аналитикой
4. **Кабинет** - Управление складом и аналитика
5. **Управление** - Административные функции

## 🎨 Особенности дизайна

### Цветовая схема
- **Фон**: Графитовый (#0F1115)
- **Акценты**: Пастельные тона (mint, blue, lavender, pink)
- **Текст**: Светлый на тёмном, тёмный на пастельном

### Анимации
- Плавные переходы между экранами
- Анимации появления карточек
- Интерактивные press-эффекты
- Анимированный индикатор активной вкладки

## 📖 Основные функции

### Главный экран
- 4 KPI метрики (продажи, чеки, выручка, прибыль)
- График прибыли за неделю
- Быстрые действия для навигации

### Приёмка товара
1. Нажмите "Сканировать штрихкод" (симуляция)
2. Выберите количество товара
3. Нажмите "Принять товар"
4. Товар добавлен, история отображается ниже

### Продажа
1. Используйте поиск или категории для поиска товара
2. Нажмите на карточку товара, чтобы добавить в корзину
3. В корзине настройте количество, скидку и способ оплаты
4. Нажмите "Продать"
5. Появится модальное окно подтверждения

### Кабинет
Переключайтесь между тремя вкладками:

#### Склад
- Сводка по остаткам
- Фильтрация товаров
- Группировка по категориям

#### Продажи
- Выбор периода (сегодня, неделя, месяц)
- Сводка выручки и прибыли
- Список продаж по дням

#### Товары
- Полный список товаров
- Редактирование и архивация
- Добавление новых товаров

### Управление
Административные функции:
- Редактирование/отмена продаж
- Управление долгами
- Работа с резервами
- Управление платёжными картами
- Резервное копирование

## 🎨 Просмотр дизайн-системы

1. На главном экране нажмите иконку палитры (правый верхний угол)
2. Откроется полная документация дизайн-системы
3. Изучите цвета, компоненты, анимации
4. Вернитесь нажав "← Назад к приложению"

## 💡 Демо-функции

В правом нижнем углу находится плавающая кнопка с меню:
- **Toggle Loading** - Переключение состояния загрузки на главной
- **Design System** - Открыть дизайн-систему

## 🎯 Советы по использованию

1. **Быстрый доступ**: Используйте быстрые действия на главной для навигации
2. **Поиск**: На экране продажи активно используйте поиск и фильтры
3. **Уведомления**: Обращайте внимание на toast-уведомления в верхней части экрана
4. **Анимации**: Все взаимодействия сопровождаются плавными анимациями

## 🔍 Mock данные

Приложение использует демонстрационные данные:
- Товары: ELFBAR, HQD, VAPORESSO и др.
- Продажи: Последние 3 дня
- Резервы и долги: Примеры записей
