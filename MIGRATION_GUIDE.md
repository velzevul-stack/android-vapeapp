# 📋 Руководство по обновлению VapeStoreApp

## ⚠️ ВАЖНО: Сохранение данных при обновлении

При обновлении приложения (особенно при смене подписи APK) данные могут быть потеряны. Следуйте этой инструкции для безопасного обновления.

---

## 1. Экспорт данных перед обновлением

1. Откройте приложение VapeStore
2. Перейдите в **Кабинет** (иконка профиля внизу)
3. Нажмите вкладку **⚙️ Настройки**
4. Нажмите кнопку **"Экспортировать данные"**
5. Дождитесь сообщения "✅ Данные экспортированы"
6. Файл сохраняется в папку приложения: `Android/data/com.example.vapestore/files/VapeStoreBackups/`
7. Скопируйте файл `VapeStoreBackup_YYYYMMDD_HHmmss.json` в безопасное место (например, Downloads или облако)

---

## 2. Установка новой версии

1. Установите новый APK поверх старой версии
2. **Если возникла ошибка "Signature mismatch" (Несовпадение подписи):**
   - Это означает, что APK подписан другим ключом
   - **Вариант А:** Удалите старое приложение (⚠️ ДАННЫЕ БУДУТ ПОТЕРЯНЫ!)
   - **Вариант Б:** Установите новое приложение после удаления
   - После установки перейдите к шагу 3 для импорта данных

---

## 3. Импорт данных после обновления

1. Откройте приложение
2. Перейдите в **Кабинет** → **⚙️ Настройки**
3. Нажмите **"Импортировать данные"**
4. Выберите сохранённый JSON файл (VapeStoreBackup_*.json)
5. Дождитесь сообщения "✅ Данные импортированы"
6. Перезапустите приложение при необходимости

---

## 4. Проверка данных

После импорта проверьте:
- ✅ Количество товаров в разделе "Товары"
- ✅ История продаж в разделе "Отчёты"
- ✅ Склад в разделе "Склад"

---

## 📁 Где найти файл резервной копии

- **Путь:** `Android/data/com.example.vapestore/files/VapeStoreBackups/`
- Используйте файловый менеджер (например, "Файлы" или "Проводник")
- Перейдите: Внутренняя память → Android → data → com.example.vapestore → files → VapeStoreBackups
